Projects

  1. Escape(FPP Horror)

Engine: Unity

Platform: PC(Personal Computer)

Role: Game Developer

Genre: Survival Horror

Overview

"Escape" is a thrilling survival horror game where the player is trapped in a house with a menacing chainsaw-wielding antagonist. The goal is to navigate through the eerie environment, open crates to discover hidden codes, and use them to unlock the door and escape the house before the chainsaw man catches you.

Key Features

  • Tense Atmosphere: The game creates a gripping sense of dread as the player navigates the dark, mysterious house.
  • Puzzle Mechanics: Players must explore the environment and solve puzzles by finding codes hidden in crates to unlock the door.
  • AI-Driven Antagonist: The chainsaw man is programmed to roam the house, increasing the challenge as the player tries to avoid detection.
  • Immersive Sound Design: Audio cues and effects enhance the suspense, making the experience more immersive.

Development Highlights

  • Utilized Unity for both 2D and 3D elements to create a realistic and engaging environment.
  • Implemented AI behaviors for the antagonist to create unpredictable and dynamic challenges.
  • Designed and scripted interactive objects like crates and doors, which are central to the gameplay.


2. Flappy Bird

Engine: Unity

Platform: PC, Mobile

Role: Game Developer

Genre: Arcade

Overview

Flappy Bird 2D is a recreation of the classic, highly addictive arcade game where players must guide a bird through a series of obstacles. The objective is to keep the bird in flight by tapping the screen (or clicking the mouse) to avoid pipes and score as many points as possible.

Key Features

  • Simple, Addictive Gameplay: Easy to learn but challenging to master, keeping players engaged.
  • Smooth Controls: Responsive tap/click mechanics that ensure a fluid gaming experience.
  • Pixel Art Style: A visually appealing, retro-inspired 2D design that captures the charm of the original.
  • Endless Challenge: The game features an endless loop of obstacles, making every playthrough unique.

Development Highlights

  • Created using Unity with custom scripts for smooth and responsive bird movement.
  • Designed and implemented collision detection for obstacle interaction.
  • Utilized Unitys UI system to display the players score and game over screen.
  • Integrated basic sound effects to enhance the gameplay experience.


3. Paws&Clawss

Engine: Unity

Platform: PC, Mobile.

Role: Game Developer

Genre: Action/Adventure

Overview

"Paws&Claws" is an engaging action-adventure game where players control a clever cat on a mission to rescue chickens from a mischievous dog. The goal is to save the chickens and safely guide them back to their home, all while avoiding the dogs attempts to thwart the rescue.

Key Features

  • Unique Gameplay: Combines elements of stealth and strategy as players navigate the environment to rescue chickens.
  • Challenging Levels: Each level presents new obstacles and challenges that require quick thinking and precise movements.
  • Character AI: The dog is programmed to patrol and react to the cat's movements, adding an extra layer of difficulty.
  • Colorful Graphics: Vibrant and whimsical 2D graphics that appeal to players of all ages.
  • Progressive Difficulty: As the game advances, the dog becomes more vigilant, and the levels become more complex.

Development Highlights

  • Designed and implemented AI behaviors for the dog to create dynamic and unpredictable challenges.
  • Developed pathfinding and rescue mechanics to ensure smooth gameplay when guiding chickens to safety.
  • Utilized Unity's animation tools to bring the cat, dog, and chickens to life with expressive movements.
  • Integrated sound effects and background music to enhance the overall game atmosphere.


4. Angry Birds

Engine: Unity

Platform: Mobile

Role: Solo Developer

Genre: Puzzle/Physics-Based

Overview

This project is a self-developed clone of the popular game "Angry Birds," created as a learning exercise to deepen my understanding of game development using Unity. The goal was to replicate the core mechanics of the original game, including the physics-based projectile system and destructible environments, to gain hands-on experience with Unity's features and C# scripting.

Key Features

  • Physics-Based Gameplay: Implemented realistic physics mechanics to simulate projectile motion and interactions between objects.
  • Multiple Levels: Designed and developed several levels with increasing difficulty, each presenting unique challenges and layouts.
  • Destructible Structures: Created various structures composed of different materials that react differently upon impact, enhancing the gameplay experience.
  • Simple UI: Developed a user-friendly interface that includes score tracking, level selection, and restart options.
  • Sound Effects: Integrated sound effects to provide auditory feedback for actions such as launching the projectile and destroying structures.

Development Highlights

  • Unity Physics Engine: Gained proficiency in utilizing Unity's built-in physics engine to create realistic motion and collision effects.
  • C# Scripting: Enhanced scripting skills by writing code for game mechanics, including the slingshot mechanism, enemy behaviors, and level progression.
  • Level Design: Learned fundamental principles of level design by creating engaging and progressively challenging stages.
  • Problem-Solving: Overcame various development challenges such as optimizing performance and ensuring consistent physics behavior across different devices.
  • Asset Creation: Developed skills in creating and importing assets, including sprites and audio files, to build a cohesive game environment.
